# EduForge — AI-Powered Adaptive Learning Platform

**EduForge** is an intelligent tutoring system that combines **Bayesian Knowledge Tracing (BKT)** with **AI-generated visualizations** to create a truly personalized learning experience. No two students ever see the same content — every visualization, hint, and practice problem adapts in real-time based on each student's demonstrated understanding.

---

## What Makes EduForge Different

Most learning platforms treat all students the same. EduForge doesn't.

When a student answers a question correctly, EduForge doesn't just say "Good job" — it recalculates the student's probability of mastery using a Bayesian model, adjusts the difficulty of the next question, and generates entirely new AI visualizations appropriate for their current level. A struggling student sees step-by-step breakdowns with real-world analogies. An advanced student sees expert summaries and challenging practice problems.

**The key innovation:** The AI (Gemini 2.5 Flash) is mathematically constrained by the BKT model. A beginner student literally *cannot* be shown expert-level content, no matter what the AI would prefer to generate. This is the neuro-symbolic fusion — the statistical model (BKT) constrains the neural model (Gemini).

## Features

### For Teachers

- **Upload Presentations** — Upload any `.pptx` file and EduForge automatically extracts topics, generates subtopics, creates MCQ question banks, and builds a concept dependency graph using AI
- **Real-Time Ingestion Tracking** — Watch the 7-step AI pipeline process your presentation in real-time, with live progress updates
- **One-Click Publishing** — Review generated content, then publish with a shareable lesson ID for students
- **Class Analytics Dashboard** — See a heatmap of every student's mastery across every concept. Identify struggling students and weak topics at a glance

### For Students

- **Adaptive MCQ Questions** — Questions automatically adjust to your level. Struggling? Get easier questions with more hints. Excelling? Get harder multi-step problems
- **AI-Generated Visualizations** — 8 types of educational components generated by AI, each tailored to your current understanding:

  | Component | What It Does | When You See It |
  |-----------|-------------|----------------|
  | **StepByStep** | Breaks concepts into numbered steps with examples | When you're just starting out |
  | **HintCard** | Gentle, moderate, or direct hints | When you need a nudge |
  | **FormulaCard** | Formulas with explained variables and worked examples | When learning equations |
  | **AnalogyCard** | Connects abstract ideas to real-world experiences | When you're a complete beginner |
  | **ConceptDiagram** | Flow charts, hierarchies, and relationship maps | As understanding develops |
  | **PracticeExercise** | Problems with progressive hints and solutions | When you're ready to apply knowledge |
  | **ProofWalkthrough** | Formal proofs with step-by-step justification | When you're proficient |
  | **ExpertSummary** | Dense expert-level overviews with challenge questions | When you've mastered the topic |

- **Live Mastery Tracking** — See your understanding of each concept update in real-time as you answer questions. Watch your mastery bars fill up as you learn
- **Smart Caching** — Visualizations are cached so you can revisit concepts without waiting for regeneration

### Adaptive Learning Engine

EduForge uses 5 scaffold levels that determine what content the AI can generate:

| Level | Name | Mastery Range | Experience |
|-------|------|--------------|------------|
| 0 | **Novice** | Below 20% | Simple language, step-by-step breakdowns, real-world analogies |
| 1 | **Developing** | 20% – 40% | Guided explanations with some technical terms |
| 2 | **Approaching** | 40% – 60% | Concept diagrams, practice problems with hints |
| 3 | **Proficient** | 60% – 80% | Edge cases, multi-step challenges, formal proofs |
| 4 | **Mastered** | Above 80% | Expert summaries, Socratic questioning, competition-level problems |

## How It Works

### Teacher Flow
```
Upload PPT → AI extracts topics → AI generates MCQs → AI builds concept graph
→ Review → Publish → Share lesson ID with students → Monitor class analytics
```

### Student Flow
```
Enroll with lesson ID → Pick a subtopic → See AI visualization → Answer MCQ
→ BKT updates mastery → Scaffold level adjusts → New visualization generated
→ Repeat until mastery → Move to next subtopic
```

### The Learning Loop (What Happens Every Time a Student Answers)
1. Student selects an answer to the MCQ
2. The BKT engine runs a Bayesian probability update
3. If mastery went up or down enough to change scaffold level, the AI generates new content
4. The mastery dashboard updates in real-time
5. The next question adapts to the new level

## Getting Started

### Prerequisites

- Node.js 18+ and pnpm
- Python 3.11+ with a virtual environment
- A Google Cloud project with Firestore, Cloud Storage, and Gemini API enabled
- Firebase project with email/password auth enabled

### Quick Start

1. **Clone and install:**
   ```bash
   git clone <repo-url>
   cd EduForge-GenUI
   pnpm install
   ```

2. **Set up environment:**
   ```bash
   cp .env.example .env.local
   # Fill in your GCP project ID, Firebase config, and Gemini API key
   ```

3. **Start backend services:**
   ```bash
   bash scripts/start-all.sh
   ```

4. **Start frontend:**
   ```bash
   pnpm dev
   ```

5. **Open:** http://localhost:3000

### Services

| Service | Port | Description |
|---------|------|-------------|
| Next.js Frontend | 3000 | Web app + GenUI API route |
| API Gateway | 8000 | Auth, lessons, enrollment, analytics |
| BKT Service | 8001 | Knowledge tracing engine |
| Ingestion Service | 8003 | PPT processing pipeline |
